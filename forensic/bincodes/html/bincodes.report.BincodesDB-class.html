fetch(self, cc)

source code 

Fetch relevant information from the credit card number.

Parameters:
  • cc (str) - Credit card number.
Returns: tuple(str, str, str, str, str, str)
Some or all of the following:
  • Credit card number, sanitized with pretty.
  • Issuer entity.
  • Card brand (see brand).
  • Card industry type (see mii).
  • Country of origin.
  • Bank's customer support telephone number.

Missing fields are returned as None.
